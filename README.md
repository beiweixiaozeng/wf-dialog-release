## 🎯 核心功能

### 🔄 多账号管理

- 批量导入账号（支持 `邮箱;密码` 格式，密码中可包含分号）
- 一键切换当前活跃账号
- 账号导出到剪贴板，方便在 SSH 远程服务器粘贴导入
- 清除已耗尽账号
- 排序：按配额 / 到期时间排序
- 搜索：快速过滤账号

### � 配额实时监控

- 日配额 / 周配额剩余百分比实时显示
- 彩色血条可视化（绿 > 60% / 黄 30-60% / 红 10-30% / 暗红 < 10%）
- Section 标题栏显示所有账号总日剩余和总周剩余
- 底部状态栏显示当前账号配额，点击快速切换

### ⚡ 自动无感切换

- 配额耗尽自动切换到下一个可用账号
- 静默模式（不弹确认框）
- 自定义日配额切换阈值（0-100%）
- 设置持久化，重启不丢失

### � 工具箱

- **应用补丁 / 恢复补丁**：Windsurf 功能增强
- **重置机器码**：一键重置设备标识
- **分身管理**：创建多个 Windsurf 实例，支持自定义命名和重命名
- **卡密激活**：授权管理

### 🔍 检查更新

- 顶部一键检查新版本
- 旋转动画 + 状态栏反馈
- 无需卡密即可检查

## 🚀 安装教程

### 步骤一：下载插件

在本页面找到最新的 ` .vsix` 文件并下载

### 步骤二：安装插件

**方法 A（推荐）**：直接拖拽

1. 打开 Windsurf
2. 按 `Ctrl+Shift+X` 打开扩展面板
3. 将 `.vsix` 文件**拖入**扩展列表区域

**方法 B**：命令安装

1. 按 `Ctrl+Shift+P` 打开命令面板
2. 输入 `Install from VSIX`
3. 选择下载的 `.vsix` 文件

### 步骤三：重启 Windsurf

完全关闭后重新打开，左侧出现 **切号器** 图标
